# خطوات تشغيل مشروع Islamenc

هذا الدليل يشرح كيفية إعداد وتشغيل المشروع محلياً على جهازك.

## 1. المتطلبات الأساسية

يجب التأكد من تثبيت البرامج التالية:

- **Node.js**: (اصدار 18 أو أحدث)
- **PostgreSQL**: لقاعدة البيانات.
- **MeiliSearch**: لمحرك البحث.

---

## 2. إعداد قاعدة البيانات (PostgreSQL)

1. قم بإنشاء قاعدة بيانات جديدة باسم `islamenc`.
2. استخدم ملف `dump.sql` الموجود في الجذر لاستعادة البيانات:
   ```bash
   psql -U postgres -d islamenc -f dump.sql
   ```
   _(ملاحظة: تأكد من تعديل اسم المستخدم وكلمة المرور في ملف `.env` لاحقاً)_

---

## 3. إعداد محرك البحث (MeiliSearch)

1. قم بتشغيل MeiliSearch على جهازك (المنفذ الافتراضي `7700`).
2. تأكد من إعداد `MEILI_MASTER_KEY` في ملف البيئة (الموجود افتراضياً في ملفات المشروع).

---

## 4. إعداد وتشغيل الـ Backend (NestJS)

1. انتقل إلى مجلد الـ backend:
   ```bash
   cd backend
   ```
2. قم بتثبيت المكتبات:
   ```bash
   npm install
   ```
3. تأكد من إعداد ملف `.env` بالقيم الصحيحة (قاعدة البيانات، MeiliSearch، والمنفذ).
4. تشغيل الخادم في وضع التطوير:
   ```bash
   npm run start:dev
   ```
   _يعمل الخادم افتراضياً على الرابط: `http://localhost:4000`_

---

## 5. إعداد وتشغيل الـ Frontend (Astro)

1. انتقل إلى مجلد الـ frontend:
   ```bash
   cd frontend
   ```
2. قم بتثبيت المكتبات:
   ```bash
   npm install
   ```
3. تأكد من أن ملف `.env` يحتوي على الرابط الصحيح للـ API:
   ```env
   PUBLIC_API_URL=http://localhost:4000/api
   ```
4. تشغيل موقع التطوير:
   ```bash
   npm run dev
   ```
   _يعمل الموقع افتراضياً على الرابط: `http://localhost:4321`_

---

## ملاحظات إضافية

- تأكد من تشغيل الـ Backend قبل الـ Frontend لضمان جلب البيانات بشكل صحيح.
- في حالة وجود أي مشاكل في الاتصال بقاعدة البيانات، تأكد من صلاحيات المستخدم `postgres`.
